Designing for digital accessibility – Extra info

Want to make sure anyone can enjoy the beauty of your designs? Explore UI and interaction design principles that contribute to creating barrier-free experiences.

What you’ll learn

  • Improving readability using colour, layout and spacing
  • How responsive design supports UI customisation
  • Accessible alternatives for media and images
  • Structuring content for easy navigation
  • Designing for keyboard, pointer and voice interaction
  • Creating accessible forms

Who should attend

  • UX designers
  • Visual designers
  • Digital professionals

Knowledge level

  • Basic knowledge of UI design is required.
  • No prior accessibility knowledge is required.

Technology requirements

We use online tools such as FigJam to work together during this course. You won’t need to install or create an account to use these tools.
