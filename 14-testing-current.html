Testing web accessibility for teams

Course duration: 3 x 2-hour sessions

Who said accessibility testing is not a team sport? Learn some quick and easy tests that will uncover common accessibility issues we see in websites. This course is for any digital professional interested in accessibility testing.

What you’ll learn

  • using automated and manual testing techniques
  • making sure all tasks can be completed using a keyboard
  • using tools to check contrast and colour usage
  • Identifying different image types and checking accessible alternatives for images
  • Checking content structure is logical and meaningful
  • making sure all tasks can be completed using a keyboard
  • Checking form labels and instructions are clear and coded correctly

Who should attend

This course is great for:

  • QA testers
  • Developers
  • Digital professionals

Knowledge level

  • No prior accessibility knowledge is required.

Technology requirements

You’ll need to download and install the following software before the start of the course:

  • Axe by Deque
  • Microsoft Accessibility Insights

If there’s anything that stops you from installing these, let us know as soon as possible and we can explore other options for you.

We also use online tools such as FigJam to work together during this course. You won’t need to install or create an account to use these tools.
